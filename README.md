# CatchIt

Dieses Repository enthaelt eine Android-App im Unterordner `catchit-android`.

## Build

```bash
cd catchit-android
./gradlew assembleDebug
```

## Starten

- Projekt in Android Studio oeffnen: `catchit-android`
- App auf Emulator oder Geraet starten

## Hinweis

Die aktuelle Ausrichtung ist Android-only.
